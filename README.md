GoogleMap_Task2
Project Overview
GoogleMap_Task2 is an Android mobile application developed using Android Studio and Java. The
application integrates Google Maps to provide location-based services such as navigation, nearby
place discovery, and saving favourite locations.
Features
- Display and interact with Google Maps
- Access the user's current location
- Search for places on the map
- Navigate to nearby places
- Get directions between two locations
- Save favourite locations
- View saved locations
- User registration and account management
- Application settings
Technologies Used
- Java
- Android Studio
- Google Maps API
- Google Location Services (GPS)
- Gradle Build System
How to Run
1. Open Android Studio
2. Open the project folder
3. Allow Gradle to sync
4. Add a valid Google Maps API key
5. Run on an emulator or physical device
Purpose
This project was developed for educational purposes to demonstrate Google Maps integration and
location-based Android development.
